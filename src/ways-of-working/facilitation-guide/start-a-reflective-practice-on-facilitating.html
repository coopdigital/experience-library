If you want to start a reflective practice on facilitating:

  • make notes on how you think it went as soon as possible
  • ask people for feedback directly if you are comfortable doing that
  • watch the recording if there is one. Watching yourself back can be difficult but very useful
  • identify any areas you want to improve and how you want to do that. It could be by asking someone more experienced for a chat or reading about it
  • make notes about how you could improve your materials straight away, ideally on them. It's so easy to forget the detail.
