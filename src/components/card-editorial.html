Guidance

An editorial card gives people enough information to make a considered decision whether to click or tap. It is a good way of presenting content consistently without overloading a page with information.

Editorial cards are ideal for presenting general information on different subjects on the same page. If you are using cards to present a group of items that share common features or characteristics consider using product cards instead.

Don’t use buttons or links to other destinations within a card as this may cause confusion or uncertainty for users.

The entirety of a card should be active, to create a large, easily clickable area.

For more information see our general guide on accessibility.

Required elements

These are parts of the component that must be included:

Title

The short title should quickly tell the user what they will find on the linked page.

Image

The supporting image must relate to the content by helping a user understand the content more quickly. It must have an alt tag that is descriptive of the image content. Go to the accessibility standard for alternative text.

If the editorial card is being used in Contentful the editor can upload a large 4:3 ratio image and the CMS (Content management system) will automatically generate and add the correct sizes.

If using the code, 2 image sizes should be supplied as the component contains code that will serve the correct image depending on the capability of the browser and size of the device.

Image sizes to supply are::

  • 780 × 438 px
  • 400 × 225 px

Link

The whole card clickable and and links to a single destination. If you need to display text below an image in a column that is not clickable use an image and text laid out using the grid.

Go to the content guidelines on links.

Optional elements

These are parts of the component that can be included if needed:

Paragraph

The paragraph can be used to contain supporting information on the topic of the card.

Label

The editorial card can have an optional label to show a category or tag associated with the content or a paragraph if the title alone does not describe the content accurately.

Using an editorial card or cards on a page

Using up to 3 cards in a row

Cards should be used in horizontal columns no more than 3 using the grid. The grid helps you to adapt the number of columns the editorial card is displayed at over different screen sizes.

Example

3 editorial cards used to highlight content from Co-op Food.

Using one card in a row

Editorial cards adapt well, whether displaying 3 cards in a row or 1. For example if one card is placed in a row it will adapt to a horizontal format and fill the whole row.

Used on their own, editorial cards will reformat automatically to fill the available space.
