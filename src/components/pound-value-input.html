Guidelines

Behaviour

The pound value input behaves like a text input, but only accepts a number. We have designed the pound value input with a “£” prefix to tell the user that the information being requested by the field is an amount in pounds.

Field width

Use a pound value input that is appropriate for the amount of information someone needs to enter. Do not make it too wide or too narrow. Too wide and it might intimidate or confuse, too narrow and it might frustrate.

Mobile numeric keypad

On touch devices, the price input should trigger the numeric keypad with a decimal point.

Formatting the price

When formatting the price you should:

  • format with commas to more clearly represent a pounds as the user types
  • allow the user to enter a decimal value
  • not show an error if the user doesn't enter a decimal value but one is required, instead when the form is submitted add a value of “.00”

Required elements

Label

Pound value inputs must have a label that tells people what information to enter. Space form fields evenly and close to their associated label.

The label must be short and make it clear what you're asking for.

Prefix

The prefix on the number input helps the user understand what kind of information is required.

Make sure you use a clear label as well as the prefix. For example, if you're asking someone to enter the value of their car, your form label should say, “Car value, in pounds”.

Error messages

Use error messages with the form validation pattern

Give error messages after the user submits the form. Do not give them as someone is entering information into each input. This can cause stress and frustration.

Error messages should appear above the relevant text input. This means they are read out by assistive technology in an order that makes sense.

Go to the guidelines on form designing a form

Focus ring

The focus ring appears as a purple outline around any interactive object, including inputs, when someone selects it.

Do not remove the focus ring. Some people who use assistive technology need it to navigate our products and services.

Optional elements

Hint text

You can add optional hint text to help people understand the information they need to enter.

For example, a pound value input might have the hint text 'For example, enter 2000'.

Placeholder text

Placeholder text is text inside an input that disappears when someone selects that input.

Do not use placeholder text. It can be hard for people to remember what it said.
