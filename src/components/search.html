Example

Search component with placeholder content.

Required elements

Label and placeholder content

Although labels on input fields should usually be visible, and placeholder content avoided, the search component is an exception.

Placeholder content is the content that sits within the search box, and disappears when someone starts to enter a search term. Use placeholder content to describe where the person is searching, for example, ‘Search [website name]’.

Size

Use the search component at either:

  • 60 pixels high
  • 44 pixels high

Set the width of the search component using the grid. The grid is a flexible way to lay out components that adapt to different device sizes. This means the search component will adapt to fit your design.

Go to the guidelines on the grid.

Make sure the search component is not:

  • too wide – as people using a screen magnifiers could find it hard to locate the search button
  • too small – as people may find it hard to find and enter words into

Check that the placeholder content is not cut off on smaller screen sizes.

Optional elements

Button colour

If search is the main action someone will make on your page, change the colour of the button to a primary button.

Go to the guidelines on buttons.
