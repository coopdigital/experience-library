Guidelines

You can use editorial cards to present links to content about different subjects in a consistent way.

An editorial card should give people enough information to make an informed decision about whether to click or tap on the card.

You should make the whole card active to create a large clickable area.

An editorial card should only link to one other destination. Do not use buttons or links to any other destinations within an editorial card.

Required elements

Title

The short title should:

  • be no more than 90 characters in length
  • make it clear what the user will find on the page that the card links to

Image

The image must relate to the content.

It must have an alt tag that describes the image content. For more information, go to the accessibility standard for alternative text.

If you are creating an editorial card in our content management system Contentful, you should upload a 780 × 438 pixel width image(???). Contentful will then generate and add the required image sizes.

If you are creating an editorial card using the code, you should supply 2 image sizes:

  • 780px (width) × 438px (height)
  • 400px (width) × 225px (height)

Link

The whole card should be clickable and link to one destination.

If you need to display text below an image in a column but you do not want it to be clickable, use the grid to lay out the image and text.

Optional elements

Paragraph

If you need to include supporting information in your editorial card, you can use a paragraph.

Paragraphs should be no more than 170 characters in length.

The paragraph sits below the title.

Label

You can add a label to show a category or tag associated with the content.

This could help people to identify content that interests them.

The label sits above the title.

Using an editorial card or cards on a page

Using up to 3 cards in a row

You should not use more than 3 editorial cards in a row. You should lay these out using the grid.

The grid will help you to set the number of columns the editorial card is displayed over for different screen sizes.

Example

3 editorial cards used to highlight content from Co-op Food.

Using one card in a row

Editorial cards will adapt to fill the available space.

If you place one card in a row, it will change to a horizontal format and fill the whole row.
