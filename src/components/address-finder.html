Guidelines

If you use an address finder, make it clear that it will only find UK addresses.

Address finders can help people to find an address by entering information they choose. This could be any part of an address, including a postcode.

As people enter information, suggested results will be displayed in a list. The more information someone enters, the more the results will be narrowed down. People can then select an address from the list.

For consistency across Co-op products and services, do not change the styling of the address finder.

Manual address entry

People can also choose to enter their address manually. This helps people who have:

  • a non-UK address
  • an address that’s not listed in the finder
  • an address that is not listed in the way the person wants

If the user selects the option to enter their address manually, then the address finder will be replaced by text input fields.

They can use these inputs to enter their address or they can select a link to return to the address finder.

Label

The address finder and manual address entry must have labels that tell people what information to enter.

The labels must be short and make it clear what information you’re asking for, for example ‘Your address’.

Hint text

You can add optional hint text to help people understand the information they need to enter.

For example, ‘This will be the address that we’ll send your Membership card to.’

Placeholder text

Placeholder text is text inside an input that disappears when someone selects that input.

Do not use placeholder text. It can be hard for people to remember what it said.

Error messages

Give error messages after the user submits the form.

Error messages should appear above the address finder or manual address entry. This means they are read out by assistive technology in an order that makes sense.

Go to our guidelines on form validation.

Focus ring

The focus ring appears as a purple outline around any object, including an address finder and manual address entry when someone selects them.

Do not remove the focus ring. Some people who use assistive technology need it to navigate our products and services.
