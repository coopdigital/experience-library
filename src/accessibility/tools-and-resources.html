Testing existing products and services

Tools

WAVE

Free automated tool that checks web content for accessibility issues using the Web Content Accessibility Guidelines (WCAG).

Silktide

Free accessibility checker for Chrome which covers:

  • WCAG and Americans with Disability Act (ADA) compliance checks
  • disability simulations
  • screen reader simulator

WebAIM colour contrast checker

Free tool that displays the colour contrast ratio between the foreground and background colour. It also states whether it passes or fails Level AA and AAA requirements for both normal text and large text.

Testing your content

Tools

Hemmingway App

Free tool that visually highlights lengthy, complex sentences and common errors. The highlights are in different colours which identify different issues, for example, a yellow highlight means the sentence is too long and advises you to shorten or split it up.

Grammarly

Free tool that:

  • checks grammar and spelling
  • checks sentence structures
  • uses artificial intelligence (AI) to generate text

Resources

Co-op's content guidelines

Guidelines for how we write in Co-op, including styles and patterns, and common words and phrases we use to make Co-op content consistent and understandable.
