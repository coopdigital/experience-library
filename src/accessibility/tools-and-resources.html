Tools

  • WAVE - free automated tool that checks web content for accessibility errors
  • Webaim colour contrast checker - tool to check that colour combinations have enough contrast
  • Hemingway - an app which checks your content for long and complex sentences
  • Silktide - browser plugin for simulating disabilities
  • A decision tree for alt text from Web Accessibility Initiative
