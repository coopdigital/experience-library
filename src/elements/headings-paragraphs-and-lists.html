Guidelines

Our headings, paragraphs and lists have set font sizes.

Go to the guidelines on font sizes.

All attributes of the headings, paragraphs and lists have been designed using values from the foundations.

Only change the styling of the headings, paragraphs and lists if these guidelines allow it.

Headings

You can style headings to look different to fit your design, for example you can make a H2 look smaller. There is code to help engineers do this.

But if you change the styling of the heading, you must keep the heading attributes so your content still has a logical heading structure.

This is because assistive technology users use the headings as a way to navigate around the page.

Paragraphs

Paragraphs are set at a size to make content readable.

We also have an ‘intro paragraph’ font size that you can use to highlight introductory content at the top of the page.

Lists

Use bullet points to break up a list of items.

Use a numbered list for a logical sequence of things, like a series of tasks that must be done in a certain order.
