Guidance

A link should be descriptive of the place it’s taking the user.

In general, you should underline links. The underline should disappear when the user hovers their cursor over it.

You do not need to underline a link if:

  • there’s another visual cue that indicates that it’s a link, for example an arrow icon
  • if it’s in context, for example within a header or footer

Links should have the correct colour contrast - go to the guidelines on colours to see the colours we use for links.

The user’s cursor should change to a hand or pointer when they hover over the link.

Because link text can be read out of context, it should always tell the user where it’s taking them. Do not use ‘click here’.

Always apply the focus ring as keyboard users need it to navigate the page.
