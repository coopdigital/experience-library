Guidance

A link should always be displayed as in the example here unless it’s used on a background colour, in which case it can be black or white.

In general, a link should be underlined. The underline should be removed on hover.

You do not need to underline a link if:

  • it’s presented with another visual cue that indicates that it’s a link, for example an arrow icon
  • if it’s in context, for example within a header or footer

Links should:

  • be checked for correct colour contrast
  • use the hand/pointer to indicate that the link is clickable
  • always have the purple focus ring as this is essential for keyboard users

Go to the colours we use for links and associated states.

Accessibility

Link text can be read out of context and should always tell the user where it’s taking them — never use ‘click here’.

Find out how to write link content.

Never remove the purple focus ring for any reason. Keyboard users need the focus ring as it is the only way they can navigate the page.

For more information see our general guide on accessibility
